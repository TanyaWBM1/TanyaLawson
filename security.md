# Security Policy

## Reporting a Vulnerability
If you discover a security issue related to this website or repository, please report it responsibly.

**Preferred contact:**  
- Email: contact@tanyamlawson.com

## What to include
- A clear description of the issue
- Steps to reproduce (if possible)
- Any relevant screenshots or URLs

## Response
I will review reports as quickly as reasonably possible and take appropriate action.


Thank you for helping keep this project safe.
