# Tanya M. Lawson

This repository contains the source code for **TanyaMLawson.com** —
a speed-first, human-centered professional website built to support
employment opportunities, partnerships, and consulting work.

## Purpose

The site is designed to:
- load extremely fast
- communicate trust and clarity
- showcase systems thinking and execution
- remain accessible across devices and browsers

It is intentionally built without heavy frameworks.

## Stack

- Static HTML5
- Modern CSS (mobile-first, responsive)
- Minimal JavaScript (only where necessary)
- GitHub Pages hosting

## Design Philosophy

- Performance is a feature
- Clarity over cleverness
- Calm systems over noise
- Human judgment over automation hype

## Status

This site is actively maintained and evolves over time.
